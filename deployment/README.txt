Nastran Punch Reader (pch2csv)
==============================

Description
-----------
A simple tool to convert Nastran Punch format to Comma-Separated Values (CSV).


Example
-------
Run the command:

$ ./pch2csv.exe example.pch

It should produce 3 files:
  example_format_0.csv
  example_format_1.csv
  example_format_2.csv
